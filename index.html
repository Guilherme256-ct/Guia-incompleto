Passos para hospedar um projeto no GitHub Pages

1. Crie um repositório com o mesmo nome do seu projeto. 2. No terminal, inicialize o repositório local com git init 3. Altere o nome da branch principal para main 4. Adicione os arquivos com git add . e faça o commit com git commit -m "first commit" . 5. Conecte ao repositório remoto com: git remote add origin htpps://github.com/SEU
